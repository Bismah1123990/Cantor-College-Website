# Cantor College website

The project is a simple multipage website for a fictional college (cantor college). It was mafr using orignal HTML and CSS as part of the web dev assignment.

## Pages Included
-'index.html'(Home)
-'courses.html'(Course)
-'about.html'(About)

## Technologies Used
-HTML5
-CSS3

No CSS frameworks (e.g Tailwind) or third party templates were used.

## How to run
Open the website by running it locally and visiting the homepage:

1.Open a terminal in the project folder.
2.Run:
-'python -m http.server 8000'
3.Open your brower and go to -'http://localhost:8000/index.html'

(Alternatively you can open 'index.html' directly, but lighthouse auditing requires HTTP/HTTPS).
